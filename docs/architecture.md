# Arquitetura CaloriaCerta v3.0

## Visão Geral

Sistema de nutrição cognitiva com três modos de interação adaptativa.

## Modos de Consciência

1. **Scanner**: Decisão rápida (3 segundos)
2. **Daily**: Tracking estruturado
3. **Reflect**: Experimentos comportamentais

## Stack Tecnológica

- Next.js 14 (App Router)
- Supabase (PostgreSQL + pgvector)
- TensorFlow.js (inferência local)
- Vercel Edge Network
